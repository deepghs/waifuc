Reading Instructions

Please note that this section is an advanced guide for using waifuc. To ensure a smooth reading experience, make sure you:

  • Have sufficient knowledge of Python, including but not limited to: a good grasp of Python’s basic syntax and common programming techniques, object-oriented programming in Python, and the use of Pip.

  • Possess at least a preliminary or intuitive understanding of machine learning, computer vision, and deep learning technologies.

  • Have a basic understanding of computer algorithms; you should be able to comprehend concepts like “workflow.”

The following sections will provide insights into waifuc’s operational mechanisms, customize methods, etc. After reading, you should have the skills to engage in diverse custom developments using waifuc. Good luck!
